# خطة ترقية محرر المستندات + دعم AI

## الهدف
ترقية محرر النصوص من محرر بسيط إلى محرر متقدم يشبه Notion مع دعم AI عبر Claude API.

---

## المرحلة 1: ترقية المحرر (ميزات بدون AI)

### 1.1 تفعيل الإضافات المثبتة لكن غير مستخدمة
- **Table** - إضافة جداول مع أزرار (إضافة جدول، إضافة صف، إضافة عمود، حذف، دمج خلايا)
- **Color** - تلوين النص مع قائمة ألوان منسدلة (8 ألوان أساسية)
- **Image** - تحسين دعم الصور مع رفع من الجهاز وإدراج URL

### 1.2 إضافة Placeholder Extension
- نص وهمي "اكتب / لعرض الأوامر أو ابدأ الكتابة..." عندما يكون المحرر فاضي

### 1.3 إضافة Slash Commands (/)
- عند كتابة `/` يطلع قائمة منسدلة فيها:
  - `/عنوان` - عنوان كبير
  - `/عنوان2` - عنوان متوسط
  - `/قائمة` - قائمة نقطية
  - `/قائمة-مرقمة` - قائمة مرقمة
  - `/جدول` - إدراج جدول
  - `/صورة` - إدراج صورة
  - `/اقتباس` - اقتباس
  - `/فاصل` - خط فاصل
  - `/كود` - بلوك كود
  - `/ai` - فتح قائمة أوامر AI

### 1.4 تحسين الـ Toolbar
- تجميع الأزرار بشكل أحسن
- إضافة أزرار الجداول والألوان والصور
- إضافة زر AI بارز (أيقونة Sparkles بلون بنفسجي)

### 1.5 CSS للجداول
- تنسيق جداول TipTap بستايل احترافي (حدود، ألوان header، RTL)

## المرحلة 2: Backend - API لـ AI

### 2.1 تثبيت Anthropic SDK
```bash
npm install @anthropic-ai/sdk
```

### 2.2 إضافة endpoint في `server/routes.ts`
```
POST /api/ai/generate
```
- يستقبل: `{ prompt, content, action }`
- `action` أنواعه:
  - `improve` - تحسين النص المحدد
  - `generate` - توليد محتوى من وصف
  - `translate` - ترجمة (عربي ↔ إنجليزي)
  - `summarize` - تلخيص
  - `fix_grammar` - تصحيح إملائي ونحوي
  - `expand` - توسيع النص
  - `shorten` - اختصار النص
  - `tone` - تغيير النبرة (رسمي/ودي/تسويقي)
  - `complete` - إكمال تلقائي
  - `custom` - أمر مخصص من المستخدم
- يرجع: `{ result: string }` (HTML)
- Streaming عبر Server-Sent Events للاستجابة الفورية
- يتطلب مصادقة (isAuthenticated)
- Rate limit: 20 طلب / دقيقة

### 2.3 متغير بيئي جديد
```env
ANTHROPIC_API_KEY=sk-ant-...
```

## المرحلة 3: Frontend - واجهة AI في المحرر

### 3.1 قائمة AI العائمة (AI Command Menu)
- تظهر عند:
  - الضغط على زر AI في الـ toolbar (أيقونة ✨)
  - كتابة `/ai` في الـ slash commands
  - تحديد نص + الضغط على الزر العائم
- القائمة تحتوي:
  - **تحسين النص** - يحسّن النص المحدد
  - **إعادة صياغة** - يعيد كتابة النص بأسلوب مختلف
  - **توليد محتوى** - يكتب محتوى من وصف تدخله
  - **تلخيص** - يلخص النص المحدد
  - **ترجمة** - يترجم عربي ↔ إنجليزي
  - **تصحيح** - يصحح الأخطاء الإملائية والنحوية
  - **توسيع** - يطوّل النص بتفاصيل أكثر
  - **اختصار** - يقصّر النص مع الحفاظ على المعنى
  - **تغيير النبرة** → قائمة فرعية (رسمي / ودي / تسويقي)
  - **إكمال تلقائي** - يكمل الجملة الحالية
  - **أمر مخصص** - حقل نصي يكتب فيه أمره

### 3.2 Floating Selection Toolbar
- عند تحديد نص يظهر شريط عائم صغير فوق التحديد
- فيه: عريض، مائل، تحسين بالـ AI (زر ✨)

### 3.3 AI Response Panel
- لما AI يشتغل يظهر panel صغير تحت النص المحدد
- فيه: النتيجة + زر "استبدال" + زر "إدراج بعده" + زر "نسخ" + زر "تجاهل"
- مؤشر loading (Sparkles يدور)

### 3.4 Inline Completion
- عند الوقوف في نهاية سطر والضغط Tab أو `/إكمال`
- يظهر النص المقترح بلون رمادي خفيف
- Enter = قبول، Escape = تجاهل
