# ميزات الذكاء الاصطناعي - AI Features

> مساعد كتابة ذكي مدعوم بـ Claude API (Anthropic) مدمج في محرر المستندات

---

## نظرة عامة

يوفر مستندك مساعد كتابة بالذكاء الاصطناعي مدمج مباشرة في محرر النصوص الغني. يستخدم Claude API من Anthropic لتقديم 10+ أوامر ذكية تساعد المستخدم في الكتابة والتحسين والترجمة.

---

## الأوامر المتاحة

| الأمر | الوصف | يتطلب تحديد نص |
|-------|-------|----------------|
| **تحسين النص** (`improve`) | تحسين الأسلوب والوضوح | نعم |
| **تصحيح** (`fix_grammar`) | تصحيح الأخطاء الإملائية والنحوية | نعم |
| **ترجمة** (`translate`) | ترجمة عربي ↔ إنجليزي (كشف تلقائي) | نعم |
| **تلخيص** (`summarize`) | تلخيص النص المحدد | نعم |
| **توسيع** (`expand`) | إضافة تفاصيل وشرح أكثر | نعم |
| **اختصار** (`shorten`) | اختصار مع الحفاظ على المعنى | نعم |
| **تغيير النبرة** (`tone`) | تغيير لنبرة رسمية/ودية/تسويقية | نعم |
| **إكمال تلقائي** (`complete`) | إكمال النص من حيث توقف | لا |
| **توليد محتوى** (`generate`) | كتابة محتوى من وصف يدخله المستخدم | لا |
| **أمر مخصص** (`custom`) | أي أمر يكتبه المستخدم | اختياري |

---

## طرق الوصول

### 1. زر AI في شريط الأدوات
- أيقونة ✨ بنفسجية في أعلى المحرر
- قائمة منسدلة بجميع الأوامر
- الأوامر التي تتطلب تحديد نص تكون معطلة إذا لم يُحدد نص

### 2. تحديد نص
- تحديد نص في المحرر
- الضغط على زر AI
- اختيار الأمر المطلوب

---

## واجهة نتائج AI

عند تنفيذ أمر AI، يظهر panel أسفل شريط الأدوات:

```
┌─────────────────────────────────────────────┐
│ ✨ النتيجة:                                 │
│                                             │
│ [النص المُحسّن أو المولّد]                   │
│                                             │
│ ┌──────────┐ ┌──────────┐ ┌──────┐ ┌──────┐│
│ │ استبدال  │ │ إدراج    │ │ نسخ  │ │تجاهل ││
│ └──────────┘ └──────────┘ └──────┘ └──────┘│
└─────────────────────────────────────────────┘
```

### الأزرار
- **استبدال** — يستبدل النص المحدد بالنتيجة
- **إدراج بعده** — يُدرج النتيجة بعد النص المحدد
- **نسخ** — ينسخ النتيجة للحافظة
- **تجاهل** — يغلق panel النتيجة

## API Backend

### Endpoint (بدون بث)
```
POST /api/ai/generate
```

**الجسم:**
```json
{
  "prompt": "حسّن هذا النص",
  "content": "النص المراد تحسينه...",
  "action": "improve"
}
```

**الاستجابة:**
```json
{
  "result": "النص المُحسّن..."
}
```

### Endpoint (مع بث - Streaming)
```
POST /api/ai/stream
```

- يستخدم Server-Sent Events (SSE)
- استجابة فورية حرف بحرف
- مؤشر loading أثناء التوليد

### الحماية
- يتطلب مصادقة (`isAuthenticated`)
- Rate limiting مخصص
- `ANTHROPIC_API_KEY` مطلوب في المتغيرات البيئية

## التنبرات المتاحة (Tone)

| النبرة | الوصف | الاستخدام |
|--------|-------|----------|
| **رسمي** (`formal`) | لغة رسمية واحترافية | عقود، مراسلات رسمية |
| **ودي** (`friendly`) | لغة ودية ومقربة | رسائل شخصية، محتوى تواصلي |
| **تسويقي** (`marketing`) | لغة تسويقية مقنعة | عروض، إعلانات، محتوى تسويقي |

---

## الكشف التلقائي عن اللغة

عند استخدام أمر الترجمة:
- إذا كان النص **عربي** → يُترجم إلى **إنجليزي**
- إذا كان النص **إنجليزي** → يُترجم إلى **عربي**
- الكشف تلقائي بناءً على محتوى النص
