# نظام المستندات والتوقيع الإلكتروني

> ميزات شبيهة بـ PandaDoc — محرر مستندات متقدم + حقول قابلة للتعبئة + توقيع إلكتروني

---

## نظرة عامة

نظام المستندات في مستندك يوفر تجربة كاملة لإنشاء المستندات وإرسالها للتوقيع الإلكتروني، مع دعم نوعين من المستندات:

1. **مستند نصي** — إنشاء محتوى عبر محرر TipTap الغني مع حقول قابلة للتعبئة
2. **مستند ملف** — رفع PDF/صورة ووضع حقول النماذج بصريًا

## محرر النصوص الغني (Text Document Editor)

### الملف: `client/src/pages/text-document-editor.tsx`

### التنسيق الأساسي
- **نص:** غامق، مائل، تحتي، مشطوب، تمييز (highlight)، كود
- **عناوين:** H1, H2, H3
- **قوائم:** نقطية، مرقمة
- **محاذاة:** يمين، وسط، يسار
- **عناصر:** روابط، صور، جداول، اقتباسات، خطوط فاصلة
- **ألوان:** تلوين النص

### الحقول القابلة للتعبئة (Fillable Fields)

إضافات TipTap مخصصة تُنشئ حقول تفاعلية داخل المستند:

| نوع الحقل | الوصف | المظهر في المحرر | المظهر عند التوقيع |
|-----------|-------|-----------------|-------------------|
| `signature` | حقل توقيع | شارة بنفسجية | لوحة رسم التوقيع |
| `text` | حقل نص | شارة زرقاء | حقل إدخال نص |
| `date` | حقل تاريخ | شارة خضراء | منتقي تاريخ |
| `initials` | أحرف أولى | شارة برتقالية | حقل إدخال قصير |

**الملف:** `client/src/components/editor/fillable-fields-extension.ts`

### المتغيرات الذكية (Smart Variables)

إدراج بيانات ديناميكية تُستبدل تلقائيًا:

```
{{profile.company}}    → اسم الشركة
{{profile.name}}       → الاسم الكامل
{{profile.email}}      → البريد الإلكتروني
{{date.today}}         → التاريخ الحالي
{{date.hijri}}         → التاريخ الهجري
```

**الملف:** `client/src/components/editor/variable-extension.ts`

### مكتبة المحتوى (Content Library)

- حفظ محتوى متكرر لإعادة استخدامه
- تصنيف حسب النوع (عام، عقد، قالب)
- إدراج سريع من الشريط الجانبي

### الحفظ التلقائي

- حفظ تلقائي بعد ثانيتين من التوقف عن الكتابة (debounced)
- مؤشر حالة مرئي: "يحفظ..." / "تم الحفظ" / "تغييرات غير محفوظة"
- حفظ يدوي عبر زر أو اختصار لوحة المفاتيح

## محرر المستندات المرئي (PDF Document Editor)

### الملف: `client/src/pages/document-editor.tsx`

### الميزات
- **رفع PDF/صور** وعرضها كخلفية
- **معاينة PDF** — تحويل PDF إلى PNG عبر السيرفر (`/api/pdf-preview`)
- **وضع حقول النماذج بصريًا:**
  - النقر على نوع الحقل في الشريط الجانبي
  - النقر على الموقع المطلوب على المستند
  - تعديل الأبعاد والموقع والتسمية
- **أنواع الحقول:** نص، تاريخ، توقيع
- **خصائص كل حقل:**
  - التسمية (label)
  - القيمة الافتراضية (value)
  - العرض والارتفاع
  - رقم الصفحة
  - مطلوب/اختياري
  - ترتيب العرض

### الحفظ
- حفظ الحقول بشكل دفعي (batch) عبر `PUT /api/documents/:id/fields`
- حفظ فردي عبر `POST` و `PATCH`

## صفحة التوقيع العامة

### الملف: `client/src/pages/sign-document.tsx`

### تجربة الموقّع
1. فتح الرابط `/sign/:shareToken` (لا يتطلب تسجيل دخول)
2. عرض معاينة المستند (PDF/صورة أو HTML)
3. **للمستندات النصية:** استخراج الحقول القابلة للتعبئة وعرضها كنماذج
4. **للمستندات الملفية:** عرض الحقول فوق معاينة PDF
5. تعبئة الحقول المطلوبة
6. رسم التوقيع على لوحة react-signature-canvas
7. إدخال الاسم والبريد (اختياري)
8. إرسال التوقيع

### البيانات المحفوظة
```json
{
  "signerName": "خالد أحمد",
  "signerEmail": "khaled@example.com",
  "signatureData": "data:image/png;base64,...",
  "ipAddress": "203.0.113.42",
  "signedAt": "2026-03-13T10:30:00Z",
  "fieldValues": {
    "field-uuid-1": "قيمة الحقل",
    "field-uuid-2": "2026-03-13"
  }
}
```

### الحمايات
- التحقق من صلاحية التوكن
- منع التوقيع المتكرر (المستند الموقّع لا يمكن توقيعه مرة أخرى)
- التحقق من الحقول المطلوبة قبل الإرسال
- نص قانوني يوضح أن التوقيع الإلكتروني يعادل التوقيع اليدوي

## قوالب المستندات (Templates)

### الملف: `client/src/components/editor/editor-templates.ts`

### الفئات

| الفئة | القوالب |
|------|---------|
| **العقود** | عقد تقديم خدمات، عقد تنفيذ مشروع، عقد توظيف، اتفاقية سرية (NDA) |
| **المستندات** | مستند عام، خطاب رسمي، محضر اجتماع |
| **العروض** | عرض سعر خدمات، عرض مشروع، عرض استشارات |

### استخدام القوالب
1. تصفح القوالب في `/templates` أو `/dashboard/templates`
2. معاينة محتوى القالب
3. الضغط على "استخدام القالب" → ينشئ مستند جديد
4. التوجيه لمحرر النصوص للتخصيص

## مسارات API المتعلقة

```
# المستندات
GET    /api/documents                     # قائمة المستندات
GET    /api/documents/:id                 # مستند مع الحقول والتوقيعات
GET    /api/clients/:clientId/documents   # مستندات عميل محدد
POST   /api/documents                     # إنشاء مستند
PATCH  /api/documents/:id                 # تحديث مستند
DELETE /api/documents/:id                 # حذف مستند

# حقول النماذج
POST   /api/documents/:id/fields          # إضافة حقل
PATCH  /api/documents/:docId/fields/:id   # تحديث حقل
DELETE /api/documents/:docId/fields/:id   # حذف حقل
PUT    /api/documents/:id/fields          # حفظ جميع الحقول (دفعي)

# التوقيع (عام - بدون مصادقة)
GET    /api/documents/sign/:shareToken    # عرض المستند للتوقيع
POST   /api/documents/sign/:shareToken    # إرسال التوقيع

# معاينة PDF
GET    /api/pdf-preview?url=...           # تحويل PDF إلى PNG

# مكتبة المحتوى
GET    /api/content-library               # قائمة المحتوى المحفوظ
POST   /api/content-library               # حفظ محتوى
PATCH  /api/content-library/:id           # تحديث
DELETE /api/content-library/:id           # حذف
```

## حالات المستند

```
draft ──→ sent ──→ signed
  │
  └──── (يمكن التعديل فقط في حالة المسودة)
```

| الحالة | الوصف | الإجراءات المتاحة |
|--------|-------|-------------------|
| `draft` | مسودة قيد الإعداد | تعديل، إضافة حقول، إرسال، حذف |
| `sent` | أُرسل للتوقيع | نسخ الرابط، انتظار التوقيع |
| `signed` | تم التوقيع | عرض التوقيع، تحميل النسخة الموقّعة |

## جداول قاعدة البيانات المتعلقة

### documents
| العمود | الوصف |
|--------|-------|
| `docType` | نوع المستند: `text` أو `file` |
| `content` | محتوى HTML (للمستندات النصية) |
| `fileUrl` | رابط الملف (للمستندات الملفية) |
| `shareToken` | توكن فريد للمشاركة |
| `recipientName/Email` | بيانات المستلم |
| `status` | الحالة: draft/sent/signed |
| `clientId` | ربط بعميل (اختياري) |

### document_fields
| العمود | الوصف |
|--------|-------|
| `type` | نوع الحقل: text/date/signature |
| `x, y` | الموضع على المستند |
| `width, height` | الأبعاد |
| `page` | رقم الصفحة (للـ PDF) |
| `required` | حقل مطلوب |

### document_signatures
| العمود | الوصف |
|--------|-------|
| `signatureData` | صورة التوقيع (PNG base64) |
| `signerName/Email` | بيانات الموقّع |
| `ipAddress` | عنوان IP |
| `signedAt` | تاريخ التوقيع |
