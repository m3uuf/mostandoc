# مستندك - Mustanadak

> منصة عربية متكاملة لإدارة الأعمال والمستندات والتوقيع الإلكتروني — شبيهة بـ PandaDoc للسوق العربي

## نظرة عامة

**مستندك** هي منصة SaaS عربية بالكامل مصممة لمساعدة المستقلين وأصحاب الأعمال الصغيرة في المملكة العربية السعودية والعالم العربي على إدارة أعمالهم بشكل احترافي. تجمع المنصة بين إدارة العملاء، العقود، الفواتير، المشاريع، المستندات مع التوقيع الإلكتروني، مساعد AI للكتابة، وصفحة عرض عامة (Portfolio).

## التقنيات المستخدمة

| المكون | التقنية |
|--------|---------|
| **الواجهة الأمامية** | React 18 + TypeScript + Vite |
| **مكتبة المكونات** | ShadCN UI + Radix UI (30+ مكون) |
| **التنسيق** | Tailwind CSS (مع دعم RTL كامل) |
| **إدارة الحالة** | TanStack Query v5 |
| **التوجيه** | Wouter |
| **محرر النصوص** | TipTap (مع إضافات مخصصة) |
| **السيرفر** | Express.js 5 + TypeScript |
| **قاعدة البيانات** | PostgreSQL + Drizzle ORM |
| **المصادقة** | جلسات مخصصة + OAuth (Google, Facebook, Apple) |
| **المدفوعات** | Stripe |
| **البريد الإلكتروني** | Resend |
| **الذكاء الاصطناعي** | Claude API (Anthropic) |
| **تخزين الملفات** | Google Cloud Storage (Presigned URLs) |
| **توليد PDF** | jsPDF + html2canvas |
| **التوقيع** | react-signature-canvas |
| **الرسوم المتحركة** | Framer Motion |
| **الخط** | IBM Plex Sans Arabic |

## الميزات الرئيسية

### المستندات والتوقيع الإلكتروني (شبيه PandaDoc)
- **محرر نصوص غني** (TipTap) مع تنسيق كامل + جداول + صور + ألوان
- **مساعد AI للكتابة** — تحسين، ترجمة، تلخيص، توسيع، تصحيح، تغيير النبرة
- **محرر مرئي لملفات PDF** — وضع حقول النماذج بالسحب والإفلات
- **حقول قابلة للتعبئة** — نص، تاريخ، توقيع، أحرف أولى
- **متغيرات ذكية** — إدراج بيانات ديناميكية (اسم الشركة، التاريخ، إلخ)
- **قوالب مستندات** — 15+ قالب جاهز (عقود، مستندات، عروض أسعار)
- **مكتبة محتوى** — حفظ واستعادة محتوى متكرر
- **تدفق توقيع إلكتروني كامل** — إرسال، تعبئة، توقيع، تتبع
- **صفحة توقيع عامة** — بدون تسجيل دخول عبر رابط مشترك
- **تصدير PDF** مع هوية الشركة
- **حفظ تلقائي** مع مؤشر حالة مرئي

### إدارة العملاء (CRM)
- إضافة وتعديل وحذف العملاء
- تصنيف حسب الحالة (نشط، محتمل، غير نشط)
- بحث وتصفية متقدمة
- حماية من الحذف عند وجود بيانات مرتبطة
- عرض مستندات العميل مع حالة التوقيع

### إدارة العقود
- 3+ قوالب عقود عربية جاهزة
- تتبع حالة العقد (مسودة، نشط، مكتمل، منتهي، ملغى)
- فتح العقد في محرر النصوص الغني للتعديل
- ربط بالعميل مع القيمة والعملة
- تصدير كـ PDF مع هوية الشركة

### إدارة الفواتير
- ترقيم تلقائي (INV-0001)
- بنود متعددة مع حساب تلقائي
- ضريبة القيمة المضافة 15% (قابلة للتعديل)
- دعم SAR/USD/EUR
- تصدير كـ PDF احترافي
- تتبع حالة الدفع (مسودة، مرسلة، مدفوعة، متأخرة، ملغاة)

### إدارة المشاريع والمهام
- لوحة كانبان (قيد الانتظار، قيد التنفيذ، مراجعة، مكتمل)
- شريط تقدم المشروع
- أولويات (منخفض، متوسط، عالي، عاجل)
- مواعيد نهائية وميزانيات

### الصفحة العامة (Portfolio)
- صفحة عرض احترافية على `/p/:username`
- تخصيص الألوان والأنماط والشعار
- عرض الخدمات والأعمال السابقة
- نموذج تواصل مع حماية من السبام
- دعم الوضع الفاتح والداكن

### مساعد الكتابة بالذكاء الاصطناعي
- تحسين النص وإعادة صياغته
- تصحيح إملائي ونحوي
- ترجمة عربي ↔ إنجليزي
- تلخيص وتوسيع واختصار
- تغيير النبرة (رسمي، ودي، تسويقي)
- إكمال تلقائي وتوليد محتوى
- أوامر مخصصة
- استجابة بالبث المباشر (Streaming)

### الاشتراكات والمدفوعات
- 3 خطط: Starter (29 ر.س)، Pro (59 ر.س)، Business (99 ر.س)
- تجربة مجانية 14 يوم
- إدارة الاشتراك عبر Stripe

### لوحة الإدارة
- إحصائيات المنصة الشاملة
- إدارة المستخدمين (أدوار، تعليق، حذف)
- التحكم بالاشتراكات
- أداة ترحيل البيانات من Bubble.io

```bash

npm install

# تشغيل بيئة التطوير
npm run dev

# فحص الأنواع
npm run check

# بناء للإنتاج
npm run build

# تشغيل الإنتاج
npm start

# تحديث قاعدة البيانات
```
