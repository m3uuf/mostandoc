## [الإصدار الحالي] - مارس 2026

### أُضيفت
- **محرر نصوص غني** — TipTap مع تنسيق كامل (غامق، مائل، عناوين، قوائم، جداول، صور، ألوان)
- **حقول قابلة للتعبئة** — إضافات TipTap مخصصة (نص، تاريخ، توقيع، أحرف أولى)
- **متغيرات ذكية** — إدراج بيانات ديناميكية في المستندات
- **مساعد كتابة AI** — Claude API مع 10+ أوامر (تحسين، ترجمة، تلخيص، توليد...)
- **بث مباشر AI** — استجابة فورية عبر Server-Sent Events
- **محرر PDF مرئي** — وضع حقول النماذج بالنقر على ملفات PDF/صور
- **نظام توقيع إلكتروني كامل** — إرسال، تعبئة، توقيع، تتبع
- **صفحة توقيع عامة** — `/sign/:token` بدون تسجيل دخول
- **قوالب مستندات** — 15+ قالب (عقود، مستندات، عروض)
- **معرض قوالب** — صفحة `/templates` عامة
- **مكتبة محتوى** — حفظ واستعادة محتوى متكرر
- **حفظ تلقائي** — مع مؤشر حالة مرئي
- **ربط مستندات بالعملاء** — مع إرسال إشعارات بريدية
- **تصدير PDF** — مع هوية الشركة والمتغيرات المُستبدلة
- **لوحة إدارة** — إحصائيات + إدارة مستخدمين + اشتراكات
- **إدارة أدوار المستخدمين** — user/admin/superadmin
- **أداة ترحيل بيانات** — من Bubble.io
- **معاينة PDF** — تحويل PDF إلى PNG للعرض
